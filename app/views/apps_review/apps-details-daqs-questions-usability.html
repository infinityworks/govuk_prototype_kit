Usability & Accessibility

B.6.1 - ISO 9241-210 lays out 6 key principles of User centred design (UCD). Has the development followed a user centred design process that conforms to this standard?

B.6.2 - Did the user centred design (UCD) process follow a set of well defined planned phases - for example discovery, alpha, beta, go live?

B.6.3 - Was a clear user demographic defined at the outset of app development?

B.6.4 - Was a representative/suitable sample of this user demographic engaged with throughout the user centred design process?

B.6.5 - Were used needs captured/confirmed through the engagement of users?

B.6.6 - Were the used needs recorded in a clear technology neutral format for example in user stories?

B.6.7 - For each of the user needs were clear user acceptance criteria defined?

B.6.8 - Were the user needs and user stories the basis upon which the apps were developed?

B.6.9 - Were early versions of the app evaluated with a sample of the user demographic?

B.6.10 - Were early versions of the app evaluated more than once with a sample of the user demographic?

B.6.11 - Before release was the app evaluated with a representative sample of the user demographic?

 

Status: Unapproved
